# Business Analytics SQL Project

## Project Overview
This project focuses on analyzing an e-commerce business dataset using SQL to extract meaningful insights related to sales performance, customer segments, product categories, and discount impact on profitability.

The analysis was performed using MySQL and includes advanced SQL concepts such as CTEs, Window Functions, and Views.

## Dataset Features
The dataset contains transactional sales data including:

- Order ID
- Customer ID
- Order Date
- Region
- Product Category
- Customer Segment
- Quantity
- Unit Price
- Discount Rate
- Revenue
- Cost
- Profit
- Payment Method

## SQL Concepts Used

- Aggregations (SUM, AVG)
- GROUP BY
- Window Functions (RANK)
- Common Table Expressions (CTE)
- Views
- Data filtering and sorting

## Key Business Insights

1. Electronics, Beauty, and Sports categories generate the highest revenue.
2. The North region contributes the highest revenue, though regional performance is fairly balanced.
3. Corporate customers rank highest in revenue contribution among customer segments.
4. Debit Card is the most frequently used payment method by customers.
5. Higher discount rates are associated with lower average profits.

## Tools Used

- MySQL
- MySQL Workbench

This project demonstrates how SQL can be used to analyze business datasets and extract actionable insights.
